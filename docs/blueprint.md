# **App Name**: SYNCA - Security Alert

## Core Features:

- Emergency Alert Activation: One-tap panic button with customizable alert levels, automatically shares GPS location and streams background audio to security teams using Supabase.
- Decoy Interface: Fake home screen triggered by a duress password, silently activates emergency alerts without visibly disabling the app, providing a safety net under duress.
- Security Team Dashboard: A real-time interface displaying alerts, location tracking on Google Maps, and audio streams from the user's device, plus multi-channel communication tools for effective coordination.
- Predictive Risk Assessment: AI-powered tool that analyzes event data, social media activity, and news feeds to identify potential threats, providing preemptive risk scores and customized security recommendations. The LLM will decide when or if to include this tool's data in a briefing.
- Wearable Device Integration: Connects with smartwatches via the Web Bluetooth API, enabling gesture-based alerts and heart rate monitoring for automatic stress detection to improve responsiveness in critical situations.
- Secure Communication: Direct messaging, video calling via WebRTC, and push-to-talk features facilitate immediate contact with security teams, using end-to-end encryption to safeguard sensitive communications.
- Automated Incident Reporting: Automatic generation of incident documentation post-event for detailed review and future security enhancements to improve protocols. Stored securely using Supabase file storage.

## Style Guidelines:

- Primary color: Deep blue (#1E40AF) to convey trust and professionalism.
- Background color: Light gray (#F8FAFC), subtly tinted the same hue as the primary, creates a clean, professional background.
- Accent color: Emergency red (#DC2626) for critical alerts and danger states; sharply contrasts with the background and primary color.
- Headings: 'Inter' Bold sans-serif for clear and legible titles.
- Body text: 'Inter' Regular sans-serif ensures readability.
- Interface elements: 'Inter' Medium for buttons and navigation to balance prominence and readability.
- Minimalist iconography with universally understood symbols for quick recognition in stressful situations.
- Prioritize accessibility and speed with a clean, professional interface. Focus on large, easily-tappable buttons and high contrast elements.
- Subtle animations to confirm actions without being distracting.
